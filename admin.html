How to add or update jobs

  1. In Excel, keep only the rows you want to add or update — or upload the whole sheet
  2. Go to File → Save As and choose CSV (Comma delimited)
  3. Enter the password above
  4. Choose the CSV file you just saved
  5. Click Upload — new jobs are added, existing jobs are updated, nothing else is touched
